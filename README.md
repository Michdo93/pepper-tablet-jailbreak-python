# pepper-tablet-jailbreak-python

This Python code is written in `pynaoqi 2.5.10.7` and can be used as an alternative to [jailbreaking with Choregraphe](https://github.com/Michdo93/pepper-tablet-jailbreak).

## Usage

You can use it by specifying the IP address of the Pepper.

```
python tablet_jailbreak.py <pepper_ip>
```

As example

```
python tablet_jailbreak.py 192.168.0.41
```
